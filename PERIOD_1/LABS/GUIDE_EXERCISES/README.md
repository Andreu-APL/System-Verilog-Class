# Guide Exercises
## Structure

*   **[Prob1_SumSub](Prob1_SumSub)**: 4-bit Adder/Subtractor with BCD Display (Combinational).
*   **[Prob2_CounterUpDown](Prob2_CounterUpDown)**: Up/Down Counter with Variable Limit (Sequential).
*   **[Prob3_SeqDetector](Prob3_SeqDetector)**: Sequence Detector "1011" (FSM).
*   **[Prob4_Multiplier](Prob4_Multiplier)**: 2-bit Multiplier (Combinational).
*   **[Prob5_Comparator](Prob5_Comparator)**: Magnitude Comparator with Alarm (Combinational).
*   **[Prob6_ShiftReg](Prob6_ShiftReg)**: Shift Register with Parallel Load (Sequential).
*   **[Prob7_ROM](Prob7_ROM)**: ROM Reader with Counter (Memory).
*   **[Prob8_Chronometer_01s](Prob8_Chronometer_01s)**: Precision Chronometer (0.1s) (Sequential).
*   **[Prob9_Lock](Prob9_Lock)**: Electronic Lock with Attempts (FSM).
*   **[Prob10_VGA_Bars](Prob10_VGA_Bars)**: VGA Solid Color Generator (VGA).

## How to Use

Each folder contains the Verilog source files. You can compile them in Quartus Prime or simulate them using Icarus Verilog.

Common helper modules like `BCD.v` and `clk_divide.v` have been copied into each project folder for self-containment.
